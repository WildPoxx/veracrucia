## Skills do projeto

Use `veracrucia-authorial-style` para escrita, reescrita, voz, estilo, anti-artificialidade e calibracao de prosa.

Use `veracrucia-scene-planner` para planejar cenas antes de redigir: POV, funcao dramatica, promessa, progresso, payoff, risco de revelacao e materialidade.

Use `veracrucia-editorial-diagnostics` para parecer critico, diagnostico de capitulo/cena, continuidade, funcao narrativa, qualidade literaria e prioridades de revisao.

Use `veracrucia-reference-decoupage` para analisar autores de referencia como mecanismos formais transferiveis, sem imitar superficie de estilo.

Use `veracrucia-context-router` antes de tarefas que dependam de canon, estado atual do projeto, busca local em `C:\Users\amari\source\Veracrucia`, busca em GitHub/Drive, arquivos anexados, fonte de verdade ou conflito entre materiais.

Use `veracrucia-repetition-control` quando houver risco de repeticao literal, estrutura frasal recorrente, enumeracao automatica, cadencia de IA, one-liner facil, reaproveitamento de metafora, nome repetido ou perda de diferenca entre POVs.
