# Veracrucia Reference Decoupage

## Purpose

Use this skill to read fiction references as craft material: identify what a text does, how it does it, why it works, what Veracrúcia can learn from it, and what must not be copied.

This skill analyzes the reference. Use `veracrucia-authorial-style` when applying lessons to Mario's prose.

## Core Rule

Never produce generic admiration. Every claim must connect:

1. textual observation;
2. formal mechanism;
3. effect on the reader;
4. transferable principle;
5. warning against imitation or misuse.

## Source Discipline

When reading copyrighted fiction:

- do not reproduce long passages;
- use short excerpts only when necessary;
- rely mainly on paraphrase, location markers and formal description;
- distinguish what the text demonstrably does, what is interpretation, what Mario might adapt and what should not be imported.

If analysis is based on selected chapters or scenes, say so.

## Workflow

### 1. Define The Reading Target

Identify:

- work and edition/file;
- scope: broad, scene-specific, style-specific, structure-specific or worldbuilding-specific;
- what Mario wants from the reference;
- what Veracrúcia needs from the analysis.

### 2. Establish A Working Question

Convert interest into a precise question, such as:

- How does this author make exposition feel like action?
- How does the city become a narrative machine?
- How does the impossible become socially plausible?
- How does dialogue create pressure without becoming theatrical?
- How does the fantastic carry social critique without explaining allegory?

### 3. Read At Two Scales

- **Macro:** structure, plot engine, genre contract, promise/progress/payoff, social field, institutions, world logic.
- **Micro:** diction, syntax, rhythm, paragraphing, image, motif, focalization, information control, dialogue, transition.

Do not let summary replace evidence.

### 4. Build The Technique Matrix

For each useful technique, record:

- **Observation:** what appears on the page.
- **Mechanism:** what formal choice creates the effect.
- **Effect:** what the reader experiences or understands.
- **Context:** why it works in that author's project.
- **Transfer:** what Mario can learn.
- **Adaptation:** how Veracrúcia would use it differently.
- **Risk:** what would become derivative, false, bloated or anti-Mario if copied.

## Relevant Axes

Use only the axes that matter to the task:

- narrative structure;
- narration and focalization;
- scene engineering;
- information management;
- world as system;
- weird or fantastic logic;
- prose texture;
- dialogue and discourse;
- ideological mechanics;
- negative lessons.

## Veracrucia Adaptation

When adapting lessons:

- preserve concrete action as scene spine;
- embody exposition in work, institution, document, body, route, machine, screen, weather or architecture;
- keep allegory submerged;
- treat paranormal as perceptual de-alienation of an already-present layer;
- use 4E: Enquadrar, Encarnar, Encadear, Ecoar;
- use 4V: Voz, Valor, Vetor, Volume;
- use Campo, Arquivo, Rastro for mysteries and weird history;
- use Lógica Submersa for paranormal systems and allegory;
- do not import surface diction if it fights Mario's prose.
