# Veracrucia

Repositório público de coordenação técnica e editorial do projeto autoral **Veracrúcia**, de Mario Bastos.

Este repositório não deve ser tratado como arquivo integral do manuscrito. A função inicial dele é manter:

- regras de colaboração com IA;
- instruções para Claude e outros assistentes;
- skills de projeto em formato legível por Claude Code;
- documentação de cobertura entre o repositório público e o vault local de escrita.

## Escopo

Veracrúcia é um projeto literário em andamento. Por isso, textos de capítulo, bíblia narrativa, notas sensíveis, referências de terceiros, PDFs, imagens pesadas, backups e materiais editoriais privados não entram no Git comum por padrão.

O conteúdo público deste repositório deve orientar o trabalho sem expor o corpus autoral completo.

## Arquivos principais

- `AI_COLLABORATION.md`: postura, limites criativos e regras de colaboração com IA.
- `CLAUDE.md`: ponto de entrada para Claude.
- `docs/ai/CLAUDE_SKILLS_INDEX.md`: espelho visível das skills do projeto.
- `docs/ai/PROJECT_KNOWLEDGE_COVERAGE.md`: o que Claude deve enxergar pelo GitHub e o que depende do vault local.
- `.claude/skills/`: skills executáveis para Claude Code.

## Regra central

Mario Bastos é o autor e diretor criativo. Assistentes podem diagnosticar, propor, organizar e redigir sob orientação, mas não devem inventar cânone, destino de personagem, passado narrativo ou decisão estética como se fossem fatos do projeto.
