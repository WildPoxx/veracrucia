## Skills do projeto

Use `veracrucia-authorial-style` para escrita, reescrita, voz, estilo, anti-artificialidade e calibração de prosa.

Use `veracrucia-scene-planner` para planejar cenas antes de redigir: POV, função dramática, promessa, progresso, payoff, risco de revelação e materialidade.

Use `veracrucia-editorial-diagnostics` para parecer crítico, diagnóstico de capítulo/cena, continuidade, função narrativa, qualidade literária e prioridades de revisão.

Use `veracrucia-reference-decoupage` para analisar autores de referência como mecanismos formais transferíveis, sem imitar superfície de estilo.
